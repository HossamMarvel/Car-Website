Car Showroom Website

A full-featured car showroom web application built with React, TypeScript, and Tailwind CSS, using Supabase for backend services.

Features

* Browse available cars
* View car details
* Calculate down payment
* Calculate monthly installments
* Responsive design for all devices

 Tech Stack

* React
* TypeScript
* Tailwind CSS
* Supabase

 Purpose

This project was built to practice building real-world web applications using modern front-end technologies and backend integration.

 ▶️ How to Run

1. Install dependencies:

bash
npm install


2. Run the project:

bash
npm run dev
